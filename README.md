# denvo-backend

## Информация
[denvoRu/Backend](https://github.com/denvoRu/Backend) - это бекенд приложение для приложения УрФУ - Оценка пар

## Стек
* Fastapi
* SQLAlchemy
* Pydantic
* Uvicorn
* DDD


## Запуск

### Установка Зависимостей
```
pip install -r requirements.txt
```
### Исполнение
```python
python main.py
```
